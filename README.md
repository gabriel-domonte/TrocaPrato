## TrocaPrato

TrocaPrato é uma ferramenta desenvolvida em Python Flask com o objetivo de amenizar a monotonia alimentar de pessoas que fazem acompanhamento nutricional. 

A ferramenta funciona de maneira simples: o usuário informa o nome e a porção em gramas do alimento que ele deveria consumir, e o nome do alimento que ele deseja consumir. A ferramenta então retorna um comparativo do valor nutricional desses dois alimentos.
